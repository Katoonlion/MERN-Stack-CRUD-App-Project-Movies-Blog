# 🍿 Popcorn Picks

![Screenshot](https://img1.pic.in.th/images/Project_03_screen_shot.jpg)

## Description

Popcorn Picks is a movie review blog application where users can share their thoughts on movies and discover reviews from others.

Users can create, edit, and delete their own reviews, as well as browse reviews posted by other users. 

This project was built to practice full-stack development, including authentication, CRUD functionality, and user-based authorization.
